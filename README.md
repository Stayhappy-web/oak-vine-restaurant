# Oak & Vine Restaurant Website

A modern, responsive restaurant website built with React and Vite.

## Features

* Responsive design for desktop, tablet, and mobile
* Elegant luxury restaurant branding
* Interactive reservation system
* EmailJS integration for reservation notifications
* Customer reservation confirmation emails
* Featured dishes section
* About section
* Contact & opening hours section
* Smooth scrolling navigation
* Custom Oak & Vine logo

## Technologies Used

* React
* Vite
* JavaScript
* CSS3
* EmailJS

## Live Demo

https://oak-n-vine-restaurant.vercel.app/

## Installation

Clone the repository:

```bash
git clone https://github.com/Stayhappy-web/oak-vine-restaurant.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
